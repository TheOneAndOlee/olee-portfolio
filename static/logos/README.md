# Logo Files

Place your social media logo PNG files in this folder with the following names:

- `github.png` - GitHub logo
- `linkedin.png` - LinkedIn logo
- `email.png` - Email icon
- `twitter.png` - Twitter/X logo

Or any other social platforms you want to add. Update the `contacts` array in `src/routes/contact/+page.svelte` to match your logo filenames and links.

## Recommended Size
- PNG files should ideally be square (e.g., 512x512px or 1024x1024px)
- The component will automatically scale them to fit
